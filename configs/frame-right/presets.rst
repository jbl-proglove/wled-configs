Presets for `frame-right` controller
====================================

This page describes the most important WLED presets for the `frame-right` controller.

-------------------

  - Start LED: `0`
  - Stop LED: `22`
  - Offset: `0`
  - Grouping: `1`
  - Spacing: `0`
  - Reverse direction: `true`
  - Mirror effect: `false`

- Segment 2: `indicator`
  - Start LED: `22`
  - Stop LED: `23`
  - Offset: `0`
  - Grouping: `1`
  - Spacing: `0`
  - Reverse direction: `false`
  - Mirror effect: `false`

- Segment 1: `vertical`
  - Start LED: `23`
  - Stop LED: `58`
  - Offset: `0`
  - Grouping: `1`
  - Spacing: `0`
  - Reverse direction: `false`

Sound settings of controller ``frame-right``
============================================

This page describes the sound settings of the controller.


Sound input settings
--------------------

- Squelch: ``6``
- Gain: ``60``


Sound input pins
----------------

- Microphone type: ``Generic I2S``
- I2S SD pin: ``32``
- I2S WS pin: ``15``
- I2S SCK pin: ``14``
